Déclaration
d'accessibilité

Captiz s'engage à rendre son site accessible à tous, conformément à l'article 47 de la loi n°2005-102 du 11 février 2005.

Partiellement conforme — RGAA 4.1
01 — Conformité

État de conformité

Le site captiz.fr est partiellement conforme avec le Référentiel Général d'Amélioration de l'Accessibilité version 4.1. Les non-conformités sont listées ci-dessous.

02 — Non-conformités

Contenus non accessibles

  • Les animations canvas (arrière-plan interactif) sont purement décoratives et ne sont pas restituées par les lecteurs d'écran.
  • Certaines transitions peuvent affecter les personnes sensibles au mouvement — un media query prefers-reduced-motion les désactive.
  • Certains textes secondaires peuvent présenter un contraste insuffisant selon les critères WCAG 2.1 AA.
03 — Technologies

Technologies utilisées

  • HTML5
  • CSS3
  • JavaScript — Canvas 2D API
04 — Tests

Environnement de test

  • Chrome + NVDA (Windows)
  • Safari + VoiceOver (macOS)
  • Firefox + JAWS (Windows)
05 — Contact

Signaler un problème

Si vous rencontrez un obstacle d'accessibilité, contactez-nous à hello@captiz.fr. Nous nous engageons à répondre sous 2 jours ouvrés.

En l'absence de réponse satisfaisante, vous pouvez saisir le Défenseur des droits.
