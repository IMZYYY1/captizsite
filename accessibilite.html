Déclaration d'accessibilité

Partiellement conforme — RGAA 4.1

Captiz s'engage à rendre son site internet accessible conformément à l'article 47 de la loi n°2005-102 du 11 février 2005. Cette déclaration d'accessibilité s'applique au site captiz.fr.

État de conformité

Le site captiz.fr est partiellement conforme avec le Référentiel Général d'Amélioration de l'Accessibilité (RGAA) version 4.1. Les non-conformités et dérogations sont listées ci-dessous.

Contenus non accessibles

Les contenus listés ci-dessous ne sont pas accessibles pour les raisons suivantes :

  • Les animations canvas (arrière-plan interactif) ne sont pas restituées par les lecteurs d'écran. Elles sont purement décoratives et sans impact sur la compréhension du contenu.
  • Certaines transitions d'animation peuvent être gênantes pour les personnes sensibles au mouvement. Un media query prefers-reduced-motion désactive les animations pour ces utilisateurs.
  • Les contrastes de certains textes secondaires peuvent être insuffisants selon les critères WCAG 2.1 AA.

Technologies utilisées

  • HTML5
  • CSS3
  • JavaScript (Canvas 2D API)

Environnement de test

Les tests ont été effectués sur les navigateurs et technologies d'assistance suivants :

  • Chrome / NVDA (Windows)
  • Safari / VoiceOver (macOS)
  • Firefox / JAWS (Windows)

Retour d'information et contact

Si vous n'arrivez pas à accéder à un contenu ou à un service, vous pouvez contacter le responsable du site pour être orienté vers une alternative accessible ou obtenir le contenu sous une autre forme.

Email : hello@captiz.fr

Nous nous engageons à répondre dans un délai de 2 jours ouvrés.

Voie de recours

Si vous constatez un défaut d'accessibilité vous empêchant d'accéder à un contenu ou une fonctionnalité du site, et que vous nous le signalez sans obtenir de réponse satisfaisante, vous êtes en droit de faire parvenir vos doléances ou une demande de saisine au Défenseur des droits : defenseurdesdroits.fr.

Date d'établissement

Cette déclaration a été établie le 20 juin 2025.
